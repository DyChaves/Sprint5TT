# Sprint5TT

Este projeto é uma aplicação web desenvolvida com Streamlit para visualizar dados de anúncios de venda de veículos.

O aplicativo permite:
- visualizar um histograma da coluna `odometer`;
- visualizar um gráfico de dispersão entre `odometer` e `price`;
- interagir com os gráficos por meio de botões.

Aplicação publicada:
https://SEU-APP.onrender.com
